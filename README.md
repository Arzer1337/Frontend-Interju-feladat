# Feladatkezelő (TODO) (React + VITE)

## Futtatáshoz:
1. Klónozd le a repositoryt a saját gépedre: `git clone [A_TE_REPO_URL_CIMED]`
2. Nyiss egy terminált a projekt mappájában.
3. Telepítsd a szükséges csomagokat: `npm install`
4. Indítsd el az alkalmazást: `npm run dev` 

## Technológiák:
* React (Hooks: useState, useEffect)
* CSS3 (Flexbox, Media Queries)
* JavaScript (ES6+)
* LocalStorage (az adatok böngészőben való mentéséhez)

## Továbbfejlesztési lehetőség:
* Drag&Drop módszer a feladatok sorbarendezéséhez
* Tailwind használat,szebb design
* Progressbar fejlesztése
* Button designs

## Felmerült problémák,nehézségek:
* Light/Dark mode kialakítása

## Design koncepció:
* Dark Theme: Visual Studio Cyberpunk theme / (személyes kedvenc theme)
* Light Theme: Alapértelmezett Visual Studio Light theme
